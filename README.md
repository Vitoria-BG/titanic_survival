## Titanic - Machine Learning from Disaster 🚢

This is my first Machine Learning project, using the classic Titanic dataset from **Kaggle**. The goal is to predict which passengers survived the shipwreck. 
The focus of this study is to practice the entire Data Science workflow, from initial data inspection to training a predictive model.

### Tools and Libraries
I used Python and the following libraries:
- **Numpy & Pandas**: Data manipulation and cleaning.
- **Matplotlib & Seaborn**: Data visualization (EDA).
- **Scikit-Learn**: Implementing Machine Learning models.

###  What I've Done So Far:
- **Data Cleaning**: Handled missing values.
- **Feature Engineering**:
    - AgeGroup: Grouped continuous ages into categorical life stages.
    - FamilySize: Combined siblings/spouses and parents/children into a single family metric.
    - TitleGroup: Extracted social status from names to help fill missing data.
- **Exploratory Data Analysis (EDA)**: Analyzed survival rates based on passenger class, sex, titles and family context.
